Display Process in for loop

Description

Display running process in for loop.

Usage

disProBar(k, NN)

Arguments

k

a positive integer, current value of loop index.

NN

a positive integer, total times of repetition.

Value

No velue returned.

Examples

  for(i in 1:1000){
  i <- i+1
  disProBar(i, 1000)
  }
